# Assets Folder

Coloca aquí tu icono principal de **1024x1024 píxeles** en formato PNG.

**Nombre del archivo:** `icon.png`

Una vez que tengas el icono aquí, ejecuta:

```bash
npx react-native set-icon --path ./assets/icon.png --platform all
```

Esto generará automáticamente todos los tamaños necesarios para iOS y Android.
